# SocNavGym: A Reinforcement Learning Gym for Social Navigation

## 🎯 Objetivos del Proyecto
El propósito principal de este proyecto es proporcionar un entorno de simulación avanzado para el desarrollo y evaluación de agentes robóticos socialmente competentes. Los objetivos específicos incluyen:

* **Desarrollar un entorno ligero y rápido:** Facilitar el entrenamiento de modelos de Deep Reinforcement Learning (DRL).
* **Simular escenarios sociales complejos:** Generar dinámicamente interacciones del mundo real, incluyendo multitudes y relaciones humano-objeto.
* **Integrar recompensas basadas en el comportamiento humano:** Implementar una función de recompensa guiada por datos (SNGNN-v2).
* **Estandarizar la evaluación:** Proveer métricas robustas para comparar el rendimiento de los algoritmos.
